# V2.1 — GitHub Secrets + Pages Setup

## Step 1 — Enable GitHub Pages

1. Go to `https://github.com/AngryWatership/gas-sim-pro/settings/pages`
2. Source: **GitHub Actions** (not branch)
3. Save

---

## Step 2 — Create GCP service account for GitHub Actions

```bash
# Create service account
gcloud iam service-accounts create github-actions \
  --display-name="GitHub Actions"

# Grant required permissions
gcloud projects add-iam-policy-binding $PROJECT_ID \
  --member="serviceAccount:github-actions@${PROJECT_ID}.iam.gserviceaccount.com" \
  --role="roles/storage.objectAdmin"

gcloud projects add-iam-policy-binding $PROJECT_ID \
  --member="serviceAccount:github-actions@${PROJECT_ID}.iam.gserviceaccount.com" \
  --role="roles/bigquery.dataViewer"

gcloud projects add-iam-policy-binding $PROJECT_ID \
  --member="serviceAccount:github-actions@${PROJECT_ID}.iam.gserviceaccount.com" \
  --role="roles/bigquery.jobUser"

gcloud projects add-iam-policy-binding $PROJECT_ID \
  --member="serviceAccount:github-actions@${PROJECT_ID}.iam.gserviceaccount.com" \
  --role="roles/cloudfunctions.viewer"

# Download key
gcloud iam service-accounts keys create /tmp/github-actions-key.json \
  --iam-account="github-actions@${PROJECT_ID}.iam.gserviceaccount.com"

# Print key for copying into GitHub Secrets
cat /tmp/github-actions-key.json
```

---

## Step 3 — Add GitHub Secrets

Go to `https://github.com/AngryWatership/gas-sim-pro/settings/secrets/actions`

Add these secrets (New repository secret):

| Secret name | Value |
|---|---|
| `GCP_SERVICE_ACCOUNT_KEY` | full contents of `/tmp/github-actions-key.json` |
| `GCP_PROJECT_ID` | `gas-sim-pro-ii` |
| `GCS_BUCKET` | `gas-sim-pro-ii-gas-sim-data` |
| `VITE_GCS_REGISTRY_URL` | `https://storage.googleapis.com/gas-sim-pro-ii-gas-sim-data/model_registry.json` |
| `VITE_CLOUD_RUN_URL` | your Cloud Run URL (from `gcloud run services describe gas-sim-api --region=us-central1 --format='value(status.url)'`) |
| `VITE_INGEST_URL` | leave blank for now (filled in V2.5) |
| `VITE_INGEST_TOKEN` | leave blank for now (filled in V2.5) |
| `VITE_COLAB_NOTEBOOK_URL` | `https://colab.research.google.com/github/AngryWatership/gas-sim-pro/blob/main/notebooks/train.ipynb` |

## Step 4 — Delete the local service account key

```bash
rm /tmp/github-actions-key.json
```

Never commit this file. It is already in .gitignore via the `*.json` pattern — but double-check:

```bash
grep "github-actions-key" .gitignore || echo "add it manually"
```
